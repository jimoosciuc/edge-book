# 8.3.8 Distributed Cloud

Distribute Cloud 介绍

* k8s和openstack云的异构分布
* 中心云（系统控制）
* 主机共享服务
* 系统范围的基础设施编排功能

控制地理位置处于遥远的，地理位置分散的边缘云

* 和系统控制节点通信，通过REST API
* 运行控制平面进行自主操作
* 符合OSF边缘计算组定义的架构模型

System Controller
* 容器平台集中部署在sub-cloud上
*   自动声明配置
*   sub-cloud的健康检查
*   同步用户的身份信息及密钥信息
*   为架构和应用提供集中式的docker仓库
*   统一的外部操作界面
*   配置通用应用针对共享平台的数据
*     DNS，NTP/PTP,API防火墙，SNMP，远程日志等
*   镜像缓存
